Folgende npm modules müsst ihr installieren

npm init -y //Erstellt package.json

npm install typescript ts-node @types/node --save-dev //Installiert typescript

MySQL-Installation erstmal verschieben. Ist zurzeit noch etwas Hickhack

npm install mysql  //installiert mysql

npm install nodemon ts-node -D //aktiviert nodemon

-> "npm run start:dev" eingeben, um mit nodemon zu arbeiten. Lädt den Server automatisch neu, sobald es Änderungen gibt
